Этот проект создан для измерения вольт - амперной характеристики элемента, в данном случае - системы из двух графитовых электродов и раствора электролита.
Для измерения используется установка, схема которой приведена в этой же папке. Установка подает на элемент напряжение и измеряет ток через него с помощью специального токоизмерительного резистора.
Очень подробно установка, программа и проведенные эксперименты описаны в следующих статьях:

https://vk.com/@fizmat_olymp-ochen-slozhnyi-eksperiment-chast-1
https://vk.com/@fizmat_olymp-ochen-slozhnyi-eksperiment-chast-2
https://vk.com/@fizmat_olymp-ochen-slozhnyi-eksperiment-chast-3

Схема установки доступна по ссылке:

Данные сохраняются в формате шестизначных чисел в 32-значной системе счисления. Первые два символа обозначают значение потенциала на А0, вторые два - на А1 и последние два символа - на А2.

Названия файлов выбираются по следующему принципу: сначала идет сопротивление токоизмерительного резистора, выраженное в килоомах. Для разделения целой и дробной части используется запятая. Затем идет апостраф `. После этого печатается само имя файла. Обратите внимание, что символы ; будут заменены на \n. После имени файла идет .txt.
Пример имени файла: 5,08`HER508;normal bias.txt. Оно означает, что измерялась ВАХ диода HER508 при нормальном подключении. В качестве токоизмерительного резистора использовался резистор на 5.08 килоом. При рисовке графика HER508 отобразится на первой строке, а normal bias - на второй.
